School Management System

We provide the easy method of school management for the Organizations having multiple branches either in same city or in different cities.  We provide a best way to handle your data with secure cloud services,  The application built to handle all type of information such as Student information, Faculty information, Library information etc...

Branch Manager can handle all kind of information related to a particular branch in the organization. Branch Operator also able to look at some type information in the branch or school.

Faculty can handle the all the information related to classes handled by him. Student information like attendance, assignments.

School Management Android Application

This is an Android version of School Management application.  Application allows login to Branch manager and Faculty.  So that they can do the all the operations they do in web application using smart phones, tablets.  One extra feature in this application is that it provides login option for students or their parents. They can login as student and can check attendance status, student progress, events, assignments and holidays etc..
